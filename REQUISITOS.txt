* ls - Listar arquivos:
    Requisitos:
      ✔️ ls
      ❌ la path/ 

* cd - Navegar:
    Requisitos:
      ✔️ cd nome_da_pasta
      ✔️ cd ..
      ❌ cd path 

* mkdir - Criar diretórios:
    Requisitos:
      ✔️ mkdir pasta
      ❌ mkdir "nova pasta"
      ❌ mkdir pasta1 pasta2 pasta3
      ❌ mkdir -p pai/filho/neto 

* touch - Criar arquivo:
    Requisitos:
      ⚠️ touch name
      ❌ touch a.txt b.txt c.txt ❌
      ❌ touch README ❌
      ❌ touch "arquivo com espaço.txt" ❌

* rm - Remover arquivo:
    Requisitos:
      ⚠️ rmf nome_arquivo
      ❌ rmf path

* rm - Remover diretório:
    Requisitos:
      ⚠️ rmd  nome_diretório
      ❌ rmd path

* mv - Renomear arquivo:
    Requisitos:
      ⚠️ rnf nome_antigo.txt novo_nome.txt

* mv - Renomear diretório:
    Requisitos:
      ⚠️ rnd nome_antigo novo_nome

* cp - Copiar arquivo:
    Requisitos:
      ❌ cpf arquivo.txt copia.txt
      ❌ cpf arquivo.txt path

* cp - Copiar diretório:
    Requisitos:
      ❌ cpd -r diretório diretório_copia.txt
      ❌ cpd -r diretório path

* cls, clear - Limpar terminal:
    Requisitos:
      ✔️ cls
